# FoodVerse

FoodVerse is a static, GitHub Pages-ready vegetarian recipe website built with HTML, CSS, and JavaScript.

## Features

- Luxury black and gold Apple-inspired interface
- Glassmorphism panels and animated 3D gold buttons
- Hero banner, live search, featured recipes, trending recipes, and vegetarian-only category filters
- 100 sample vegetarian recipes in `assets/data/recipes.json`
- Favorites saved with Local Storage
- Responsive layout for desktop, tablet, and mobile
- Dark mode and light mode toggle
- SEO meta tags and structured data
- No ordering system, Google Forms, WhatsApp ordering, or review forms

## GitHub Pages

Upload this folder to a GitHub repository and enable GitHub Pages. The site is ready when `index.html` is at the publishing root.
